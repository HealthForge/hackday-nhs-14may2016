# BloodFlow

Blood results visualisation from NHS Hackday 13 (May 2016)

## Demo

http://bloodflow.lfrg.uk/

## Uses

- AngularJS (https://angularjs.org/)
- D3 (https://d3js.org/)
- Smart-on-FHIR (http://docs.smarthealthit.org/)
